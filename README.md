# NRF24L01-Two-Way-Communication

## Arduino ile NRF24L01 Adaptörü Bağlantı Pinleri

| nRF24L01 Adaptörü | Arduino Uno |
|-------------------|-------------|
| VCC               | 5 V         |
| GND               | GND         |
| CS                | 7           |
| CE                | 8           |
| MOSI              | 11          |
| MISO              | 12          |
| SCK               | 13          |
